# ChargeCalc
This web app helps EV owners convert their time-based charging tariffs to a kWh-based tariff. This makes it easier to calculate how economical the EV is and to compare it with other tariffs.

You can find the website [here](https://chargecalc.streamlit.app/).

## Used Framework
This web app is built using Streamlit because it is user-friendly and quick to set up. Pandas is used to read the CSV file. Due to the framework being non-static and having issues setting up `stlite`, I resorted to hosting the website on Streamlit's community servers.
